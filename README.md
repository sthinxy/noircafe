## NOIR CAFÉ // CARDÁPIO DIGITAL

</div>

**Noir Café** é um cardápio digital interativo desenvolvido para uma cafeteria fictícia de estética minimalista, monocromática e levemente surreal.

Apesar da estética conceitual, o projeto possui funcionalidades reais de um cardápio digital: listagem de produtos, filtros, favoritos, carrinho, quiz de preferências, login fictício e envio de pedido pelo WhatsApp.

## Sobre o projeto

**Noir Café** simula um cardápio digital real que poderia ser adaptado para cafeterias, docerias, restaurantes pequenos, confeitarias ou marcas autorais.

A ideia foi unir uma interface funcional com uma identidade visual forte, criando uma experiência que não fosse apenas bonita, mas também navegável, responsiva e interativa.

O site permite que o usuário veja os produtos, filtre categorias, favorite itens, monte um pedido no carrinho, copie o resumo e envie o pedido pelo WhatsApp.

## Como funciona

O projeto possui uma lista de produtos armazenada em JavaScript. Cada produto contém informações como nome, categoria, descrição, preço, imagem e atmosfera relacionada.

Exemplo de estrutura de produto:

```js
{
  id: "latte-white-space",
  name: "Latte White Space",
  cat: "coffee",
  desc: "Latte suave com leite vaporizado e final delicado.",
  price: 12,
  img: "assets/images/latte.jpg",
  mood: "delicate"
}
```

Esses dados são usados para renderizar os cards do cardápio dinamicamente na tela.

O usuário pode adicionar produtos ao carrinho, alterar quantidades, remover itens e gerar um resumo do pedido.

O carrinho, os favoritos, o tema escolhido e o login fictício são salvos no navegador através do **LocalStorage**.

## Quiz de preferências

O projeto também possui um quiz interativo que pergunta sobre a atmosfera e as preferências do usuário.

Com base nas respostas, o sistema recomenda uma bebida, um doce ou um combo.

Exemplo de resultado:

```txt
sua atmosfera foi lida.
sua combinação ideal é Latte White Space + Cookie de Memória.
```

O usuário pode adicionar a recomendação diretamente ao carrinho.

---

## Login fictício

O projeto possui um sistema simples de login e criação de conta usando apenas **JavaScript** e **LocalStorage**.

Esse login é demonstrativo e não utiliza backend real.

Ele permite:

* criar uma conta fictícia
* entrar com usuário e senha
* manter o usuário salvo no navegador
* exibir o nome do usuário na interface
* encerrar sessão

```txt
Conta registrada na memória.
Bem-vindo(a) de volta.
```

## Carrinho e pedido

O carrinho funciona como um pequeno inventário.

O usuário pode:

* adicionar produtos
* aumentar quantidade
* diminuir quantidade
* remover itens
* limpar o carrinho
* copiar o pedido
* enviar o pedido pelo WhatsApp

Exemplo de pedido gerado:

```txt
Olá! Gostaria de fazer um pedido no Noir Café:

1x Latte White Space — R$ 12,00
2x Cookie de Memória — R$ 16,00

Total: R$ 28,00
```

## Como executar localmente

1. Clone este repositório:

```bash
git clone https://github.com/sthinxy/noircafe.git
```

2. Abra a pasta do projeto:

```bash
cd noircafe
```

3. Abra o arquivo `index.html` no navegador.

Ou execute com a extensão **Live Server** no VS Code.

---

## Como publicar no GitHub Pages

1. Envie os arquivos para um repositório no GitHub.
2. Vá em **Settings**.
3. Entre em **Pages**.
4. Em **Branch**, selecione `main`.
5. Selecione a pasta `/root`.
6. Salve.
7. Aguarde o GitHub gerar o link do site.

Link final do projeto:

```txt
https://sthinxy.github.io/noircafe/
```
