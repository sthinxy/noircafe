<h2 align="center">Como funciona | How it works</h2>

<div align="center">

<table>
  <tr>
    <td width="50%" valign="top">
      <h3 align="center">PT-BR</h3>
      <p align="center">
        O projeto possui uma lista de produtos armazenada em JavaScript. Cada produto contém informações como nome, categoria, descrição, preço, imagem e atmosfera relacionada.
      </p>
      <p align="center">
        Esses dados são usados para renderizar os cards do cardápio dinamicamente na tela.
      </p>
      <p align="center">
        O usuário pode adicionar produtos ao carrinho, alterar quantidades, remover itens e gerar um resumo do pedido.
      </p>
      <p align="center">
        O carrinho, os favoritos, o tema escolhido e o login fictício são salvos no navegador através do <strong>LocalStorage</strong>.
      </p>
    </td>
    <td width="50%" valign="top">
      <h3 align="center">EN</h3>
      <p align="center">
        The project has a product list stored in JavaScript. Each product contains information such as name, category, description, price, image and related atmosphere.
      </p>
      <p align="center">
        This data is used to dynamically render the menu cards on the screen.
      </p>
      <p align="center">
        The user can add products to the cart, change quantities, remove items and generate an order summary.
      </p>
      <p align="center">
        The cart, favorites, selected theme and fictional login are saved in the browser through <strong>LocalStorage</strong>.
      </p>
    </td>
  </tr>
</table>
