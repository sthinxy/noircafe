sobre o projeto

Noir Café Menu é um projeto de cardápio digital desenvolvido para demonstrar como uma experiência simples pode se tornar mais memorável através de uma identidade visual forte, interações claras e uma navegação pensada para o usuário.

Simula um cardápio real, adaptável para pequenos negócios — cafeterias, docerias, restaurantes autorais. Inclui carrinho funcional, filtros por categoria, quiz de preferências, sistema de favoritos e pedido via WhatsApp.
